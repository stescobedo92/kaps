## Why KAPS?

<CardGrid stagger>
  <Card title="Military-Grade Security" icon="seti:lock">
    AES-256-CBC encryption with PBKDF2 key derivation for maximum protection of your sensitive data.
  </Card>
  <Card title="Cross-Platform" icon="laptop">
    Single binary that runs natively on Windows, Linux, and macOS - no dependencies required.
  </Card>
  <Card title="Structure Preserving" icon="seti:folder">
    Maintains directory hierarchies and metadata during encryption/decryption operations.
  </Card>
  <Card title="Developer Friendly" icon="seti:godot">
    Perfect for CI/CD pipelines, version control, and automated workflows.
  </Card>
</CardGrid>

## Quick Start

```bash
# Install KAPS
cargo install kaps

# Encrypt a directory
kaps encrypt -i ./secrets -o ./encrypted -p "your-secure-password"
```
