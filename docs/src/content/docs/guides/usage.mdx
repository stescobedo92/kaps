    ```shell
    kaps encrypt \
      --input <source_directory> \
      --output <encrypted_directory> \
      --password "YourSecurePassphrase!"
    ```
    _Creates encrypted copy while preserving original structure_
  </TabItem>

  <TabItem label="Decrypt">
    ```shell
    kaps decrypt \
      --input <encrypted_directory> \
      --output <decrypted_directory> \
      --password "YourSecurePassphrase!"
    ```
    _Restores original files with metadata integrity_
  </TabItem>
</Tabs>

## üéØ Common Examples

<Steps>

1. **Basic Directory Encryption**
   
   ```shell
   kaps encrypt -i ./secrets -o ./encrypted -p "mypassword123"
   ```

2. **Verbose Decryption**
   
   ```shell
   kaps decrypt -i ./encrypted -o ./decrypted -p "mypassword123" --verbose
   ```

3. **Multiple Directory Processing**
   
   ```shell
   for dir in ./dir1 ./dir2 ./dir3; do
     kaps encrypt -i "$dir" -o "${dir}_encrypted" -p "mypassword123"
   done
   ```

</Steps>
