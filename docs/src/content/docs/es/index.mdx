## ¿Por qué KAPS?

<CardGrid stagger>
  <Card title="Seguridad de Grado Militar" icon="seti:lock">
    Cifrado AES-256-CBC con derivación de claves PBKDF2 para máxima protección de tus datos sensibles.
  </Card>
  <Card title="Multiplataforma" icon="laptop">
    Ejecutable único que funciona nativamente en Windows, Linux y macOS - sin dependencias adicionales.
  </Card>
  <Card title="Preservación de Estructura" icon="seti:folder">
    Mantiene las jerarquías de directorios y metadatos durante las operaciones de cifrado/descifrado.
  </Card>
  <Card title="Amigable para Desarrolladores" icon="seti:godot">
    Perfecto para pipelines CI/CD, control de versiones y flujos de trabajo automatizados.
  </Card>
</CardGrid>

## Inicio Rápido

```bash
# Instalar KAPS
cargo install kaps

# Cifrar un directorio
kaps encrypt -i ./secretos -o ./cifrado -p "tu-contraseña-segura"
```
